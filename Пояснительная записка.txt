			
			Яндекс Дневник - система электронного дневника на Python.

Возможности:
• Авторизация через аккаунт Яндекс (Яндекс ID)
• Все базовые функции электронного дневника (выставление/просмотр оценок, создание/изменение/просмотр урока)
• Уведомления об оценках и новых уроках через Telegram.
• Связь с учителем или учеником через Telegram.

	server.py:
Основной файл со всей системой сайта.

    telegramapi.py:
Дополнительный файл с API частью сайта.

	bot.py:
Файл со всей системой Telegram бота.

	templates:
Папка, где хранятся шаблоны для HTML страниц.
		admin - папка с шаблонами для страниц Администратора.
		student - папка с шаблонами для страниц ученика.
		teacher - папка с шаблонами для страниц учителя.

	static:
Папка, где хранятся css-стили, изображения и так далее.
